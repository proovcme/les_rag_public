# LES Roadmap v2.0

## Overview
This document outlines the roadmap for future development of the LES system, including planned features, milestones, and timelines.

## Milestones
### Completed in v2.0.1
- **Fix Metrics**: Resolved issues with SQLite queries and cursor.fetchone() to ensure accurate metrics collection.
- **Stabilize Core**: Improved stability and performance of the core proxy server and backend services.

### Planned for Future Releases
- **Enhanced Security**: Implement additional security measures, including authentication and authorization.
- **Scalability Improvements**: Optimize system architecture for better scalability and resource utilization.
- **Additional Data Formats**: Support more data formats for dataset uploads and parsing.
- **Advanced Query Features**: Enhance query capabilities to support more complex retrieval scenarios.
