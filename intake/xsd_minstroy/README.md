# XSD-схемы Минстроя (Q4)

Положи сюда:
- XSD-схемы форм (приоритет: АОСР / ЗОС / РНС / ПЗ). Именование: `<тип>_<версия-даты>.xsd`,
  напр. `aosr_2024-05-21.xsd`.
- 2-3 эталонные пояснительные записки (для W14).

Потребитель: W19.1 (версия-пин в реестре форм), W19.3 (генерация XSD-valid XML + валидация).
