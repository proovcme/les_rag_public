# BIM-модель БЦ «Банкрот» (Q11)

Положи сюда реальную модель бизнес-центра: Revit-экспорт плагина (`*.cad_bim_graph.json`) либо IFC/DXF.
Импорт: `POST /api/cad-bim/import` (файл копируется в `RAG_Content/CAD_BIM/JSON`, без UTF-8 BOM).

Потребитель: W17.3 (классификационный хребет Floor→System→Category), W6.1 (агрегатные чанки), W9 (норм↔элемент).
