# Образцы документов оператора (Q5)

По одному реальному файлу на формат, с фирменной вёрсткой (штамп/бланк сохраняются при заполнении).
- `docx/` — бланк акта, общий журнал работ (ОЖР). Метки `{{поле}}` проставит система при онбординге
  образца (или оператор заранее). Потребитель: W11.3 forms_service (template-fill).
- `xlsx/` — смета (ЛСР / экспорт Гранд-Смета), спецификация по ГОСТ 21.110. Потребитель: W20.1 (парсер
  смет), W20.2 (спецификации), W11.3.
- `pdf/` — бланк акта (AcroForm или плоский для координатной разметки). Потребитель: W11.3 (pdf-рендерер).
