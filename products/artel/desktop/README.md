# АРТЕЛЬ морда (Electron)

Десктоп-интерфейс Фабрики семейств. Общается с локальным бэкендом по HTTP
(`127.0.0.1:5057`); напрямую с Revit не разговаривает (см.
[дизайн пакета](../docs/family-factory-package.md)).

## Запуск (dev)

```bash
# 1) бэкенд (из корня репо)
uv run uvicorn products.artel.backend.server:app --host 127.0.0.1 --port 5057

# 2) морда
cd products/artel/desktop
npm install
npm start
```

## Экраны

- **Спецификации** — список; «Извлечь из техлиста (PDF)» → бэкенд тащит таблицу →
  спец (0 модели); карточка: параметры, типоразмеры, выбор архетипа, **Утвердить**,
  **Сгенерировать в Revit** (ставит задание в очередь бэкенда).
- **Задания** — очередь генерации + отчёт от плагина (статус по-операционно).
- **Настройки** — адрес бэкенда.

## Поток

техличка → (бэкенд) спец → ревью/approve → «Сгенерировать» → очередь → Revit-плагин
строит `.rfa` → отчёт. Морда живёт без облака/ЛЕС.

Упаковка (electron-builder → exe) — в составе общего установщика Inno Setup.
