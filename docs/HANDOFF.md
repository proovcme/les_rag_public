## Что сделано этой сессией

1. **Приёмка смонтированного объёма из исполнительных схем (сканов)** — `asbuilt_intake_service`
   (+`asbuilt_ocr`): render→авто-поворот→locate-then-read (vision-OCR — единственный LLM-шаг)→строки→
   журнал объёмов (pending). Чат `asbuilt`, `POST /api/field/extract-asbuilt`, CLI, бенч моделей.
2. **Форматы проектного архива + локальный OCR Tesseract(rus)** — legacy `.doc` (textutil; раньше
   тихо пустые!), `.xlsm`, картинки, `.p7m`, архивы. `RAG_OCR_BACKEND=tesseract` (бинарь, изолирован,
   читает русский — RapidOCR без кириллицы, Surya ломает transformers). converter на сканах зовёт наш
   rus-OCR, минуя eng-OCR pymupdf4llm.
3. **LES.md — файл-контекст папки (CLAUDE.md для ЛЕС)** + auto-init при индексации + реестр проектов.
4. **Ярус 2 — агент-роутер** (LLM сам выбирает инструмент) + **режимы local/cloud/mix** (один
   переключатель: чат+OCR+приёмка; чат/команда/кнопки в ⚙).
5. **ВОР v2** (декомпозиция ГОСТ 21.111: позиция→набор работ + графы) + **ETA индексации** (бар Самовара).
6. **Фиксы:** авто-заметка не хватает запросы; формат-директива GUI отдельным полем `output_directive`
   (не в роутинг/заметки/ретрив); `_load_commands` scope (UI-500); `/команды`-префикс.
