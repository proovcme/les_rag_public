## Что живёт и проверено

- **Позиционирование:** «Единый центр информации о строительном проекте» — ИИ-СОД + агент РП/ГИП (README).
- **Детерминированная агрегация по сметам/ВОР:** SQL по полному Parquet, не LLM. Кабель 3х1,5 = 15 030,72 м (бит-в-бит). `proxy/services/table_query_service.py`.
- **.xls читаются:** xlrd + конвертация в `parquet_writer`; дренаж индексатора; `RAG_PARSE_MIN_FREE_GB=4` под 9B.
- **Типизированный ретрив (ADR-12, флаг `LES_TYPED_RETRIEVAL=true`):** LLM-роутер по каталогу + кэш `doc_router_cache`. `proxy/services/doc_router.py`.
- **Индексация внешней папки по ссылке:** `POST /api/rag/index-external` + `LES_EXTERNAL_SOURCE_ROOTS`. ИД ЭОМ → 87 файлов, 0 копий.
- **Облако:** proxyapi.ru (Cloudflare/OpenRouter из РФ режутся). **Локаль:** Qwen3.5-9B-MLX-4bit.

## НЕ сделали — хотим (бэклог, по приоритету)

1. **Сверка ВОР↔КС-2↔смета↔ИД** (топ-кейс ГИП). Кирпичи готовы (table-агрегация + внешняя индексация + .xls→Parquet). Осталось: проиндексировать КС-2 + ЛС (смета) + распарсить ИД (87 файлов зарегистрированы, `parse=False`); сервис сравнения qty по позициям между типами документов с флагами расхождений. *Принцип подтверждён: ВОР=ЛС=КС-2 по кабелю совпали.*
2. **Родной вид «Сводный ВОР» в Совушке + скриншот** (GUI-first). Таблица из 93 позиций уже строится агрегацией; нужен вид на странице «Инструменты» + скриншот. Чат отдаёт суммы, не полную таблицу.
3. **Мультикласс end-to-end** — fan-out по классам (норматив/проект/письмо) + сборка ответа секциями + suggestions-чипы (диалог). Дизайн в ADR-12; нужен кросс-классовый тест-контент (сверка ИД даст его).
4. **АРТЕЛЬ — MEP-коннекторы** (`task_91f89b7d`): исполнение `ConnectorElement` в плагине; mpu 15/19 → 19/19. Сборка/запуск на Легионе — `legion-build-workflow` (память).
5. **Формирование ответа по интенту** (ADR-12 §2): «расскажи»→сжато, «собери всё»→развёрнуто, «значение»→строка. Детерминированный intent→форма, не дефолт модели.
6. **OCR сканов → Gemma 12:** GLM-OCR удалён (модель); переключить пайплайн скан-OCR на gemma4:12b.
7. **Влить `feat/les3-p1` → main** — когда контур стабилизируется; затем настоящий релиз (не pre).
