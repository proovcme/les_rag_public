## Исправлено в этом проходе (нулевой риск для верстки)

- **A1 Focus Visible** — `sovushka/styles.py`: глобальный `:focus-visible` контур (`outline: 2px var(--accent)`,
  двойная обводка для кнопок/табов на тёмном). Показывается только при клавиатурном фокусе — мышь
  разметку не двигает.
- **A2 lang=ru** — `sovushka_ng.py`: `<script>document.documentElement.lang='ru'</script>` в head.
- **A3 aria-label** — иконочные кнопки получили имена: история/закрыть/очистить чат (`pages/chat.py`),
  обновить/тема/настройки (`components/header.py`).
- **A4 reduced-motion** — `@media (prefers-reduced-motion: reduce)` глушит `sovload`, `blink`, `typing`,
  transitions.

После деплоя проверить визуально: контур фокуса по Tab виден на кнопках/полях/табах; скринридер
называет иконочные кнопки.

## TODO — требует визуальной приёмки/решения оператора

### A5 — Кликабельные `div` недоступны с клавиатуры (Critical для AT)
`cursor:pointer`-элементы, которые являются `ui.element('div')` с обработчиком клика, не получают
фокус и не реагируют на Enter/Space. Кандидаты: `.sov-session-card` (карточки сессий в истории),
`.mode-rag`/`.mode-code`, `.les-map-preset`, любые кликабельные строки в `pages/history.py`.
**Фикс:** заменить на `ui.button`/добавить `role="button" tabindex="0"` и обработку Enter/Space.
Точечно, по экранам — нужна проверка, что не ломает layout.

### A6 — Микро-кегль (High)
Размеры ниже ~12px массово: `.48rem` (`diag-node-state` ≈ 7.7px), `.54rem` (`les-fuse-cap`,
`diag-acronym-role`), `.56–.62rem` (теги, метки KPI, табы, подписи). WCAG не задаёт минимум, но
7–10px нарушает 1.4.4 (увеличение) и когнитивную доступность. **Рекомендация:** поднять пол до
**≥0.7rem (≈11px)** для метатекста и **≥0.78rem (≈12.5px)** для основного; теги/чипы — ≥0.68rem.
Риск: плотные грид-раскладки (`les-fuse-grid`, `diag-*`) могут переполниться — нужна проверка в
браузере, поэтому не трогал автоматически.

### A7 — Статус только цветом (Medium)
Точки `.dot/.dot-warn/.dot-err`, индикатор «proxy онлайн/офлайн», кольца статусов в графе — несут
смысл только цветом. **Фикс:** добавить текст/иконку/паттерн рядом (напр. «● онлайн» / «○ офлайн»,
галочка/крест), `title`/`aria-label`.

### A8 — Контраст нетекстовых элементов (Medium)
Границы панелей/полей/карточек на `rgba(138,162,184,.22–.32)` дают контраст с фоном < 3:1 — грани
панелей и инпутов плохо различимы (1.4.11). **Фикс:** поднять альфу до ~.45–.55 либо использовать
`--border` для значимых границ. Низкий визуальный риск, но менять глобально лучше с глазами.

### A9 — Семантика и навигация (Medium) 🟡
NiceGUI генерит `div`-дерево: мало лендмарков (`<main>` есть в чате, но нет `<nav>` для табов, нет
`<h1>`), нет skip-link «к содержимому», порядок заголовков не выстроен. **Фикс:** обернуть верхние
табы в `role="navigation"`, добавить видимый-при-фокусе skip-link, проставить уровни заголовков.

### A10 — Размер цели 24px (Low–Med, WCAG 2.2)
Чипы (`height:24px` — на грани), иконочные `flat dense` кнопки могут быть < 24×24. **Фикс:** гарантировать
`min-height/min-width: 24px` интерактивным целям.

### A11 — `opacity` на тексте (Low)
`.sov-muted{opacity:.78}`, плейсхолдеры `opacity:.78` поверх уже неяркого `--dim` — на светлых панелях
контраст может опускаться к 4.5:1. **Фикс:** задавать готовый цвет вместо `dim*opacity`.

## Замечания по верстке (вне строго WCAG, но «косяки UI»)
- Инлайн-стили размазаны по страницам (десятки `style="font-size:.62rem;color:var(--dim)"`) — единый
  источник правды только в `styles.py` облегчил бы и доступность, и поддержку. Кандидат на рефактор
  в утилитарные классы (`.text-meta`, `.text-dim`).
- Светлая тема протестирована меньше тёмной — отдельно прогнать контраст тегов (фон тегов захардкожен
  `rgba(...,.15)` независимо от темы).
- Проверить переполнение в `les-fuse-grid`/`diag-acronym-grid` на узких экранах (есть `@media`, но
  кегль крошечный).

## Как перепроверить
- Клавиатура: Tab по всему экрану — фокус виден везде, ловушек нет, Enter/Space активируют.
- Скринридер (VoiceOver): иконочные кнопки названы; язык — русский.
- Контраст: axe DevTools / Lighthouse на `/classic` и `/les/classic` (обе темы).
- `prefers-reduced-motion: reduce` в системе → анимации замирают.
