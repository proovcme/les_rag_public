# Outlook-плагин на Легионе, обмен с ЛЕС по SSH

Топология под твой кейс: **Outlook + Add-in — на Легионе** (Windows), **ЛЕС — на Маке** (mini).
Плагин общается с ЛЕС **по SSH-туннелю**, а не по прямому IP (как просил Олег). На Легионе плагин
шлёт на `http://localhost:8050`, и через SSH это попадает в локальный ЛЕС на Маке.

Бонус: `localhost` на Легионе снимает проблему mixed-content (Outlook desktop пускает `http://localhost`).

## Шаг 1. Поднять туннель (один из двух способов)

**A. С Мака (рекомендую — «поставил и забыл»)** — реверс-туннель:
```bash
bash tools/legion_mail_tunnel.sh         # держит Legion:8050 → ЛЕС, авто-переподключение
```
(использует ssh-алиас `legion` из `~/.ssh/config` → 10.195.146.20, user Oleg)

**B. С Легиона** — форвард-туннель (PowerShell на Легионе):
```powershell
ssh -N -L 8050:127.0.0.1:8050 mini       # mini = алиас этого Мака; держать окно открытым
```

Проверка с Легиона, что ЛЕС виден через туннель:
```powershell
curl http://localhost:8050/api/health
```

## Шаг 2. Поднять ассеты таскпейна (на Легионе)
В папке `clients/outlook_addin/` на Легионе:
```powershell
npm run serve-http        # вариант A из README.md — http-ассеты, без возни с сертификатами
```

## Шаг 3. Сайдлоад и кнопка
1. `uuidgen` → в `<Id>` манифеста (один раз).
2. Outlook → Get Add-ins → My add-ins → Custom Add-in → Add from File → `manifest.xml`.
3. Открыть письмо → лента, группа **ЛЕС** → **В ЛЕС** → таскпейн.
4. Поле «URL ЛЕС» оставить **`http://localhost:8050`** (туннель сам уведёт на Мак).
5. «Отправить в ЛЕС» → видишь, что куда уехало (КП→КАЦ, смета/док→RAG, скан→приёмка) + КАЦ-итог.

## Поток
```
Outlook (Легион) ─кнопка─▶ Add-in ─POST http://localhost:8050─▶ [SSH-туннель] ─▶ ЛЕС (Мак) /api/mail/push
                                                                                   └▶ КП→КАЦ / смета→RAG / скан→приёмка
```
Канал зашифрован SSH; ЛЕС наружу не открыт (слушает 127.0.0.1, туннель — на loopback Легиона).
