# Программа испытаний Л.Е.С. v2.0

**Документ:** Программа испытаний (ПИ)  
**Версия системы:** Л.Е.С. v2.0 Core  
**Дата составления:** 10.05.2026  
**Цель:** Подтверждение работоспособности облегчённого стека (Qdrant + FastAPI + LlamaIndex) и CRAG-пайплайна.

## 1. Общие положения
### 1.1. Объект испытаний
Локальная Единая Система v2.0 — суверенный инженерный RAG-стек без внешних зависимостей (RAGFlow, ES, MySQL, MinIO). Работает на FastAPI, Qdrant, Ollama (`qwen3:14b`, `bge-m3`).

### 1.2. Условия проведения
- Docker Desktop / Compose
- Ollama на хосте (11434)
- Mac M4 / 24 GB RAM (целевая платформа)
- Тестовые данные: PDF/DOCX нормативы, EML/MSG письма, XLSX таблицы
