# Программа испытаний Л.Е.С. v2.0

**Документ:** Программа испытаний (ПИ)
**Версия системы:** Л.Е.С. v2.0 Core
**Дата составления:** 10.05.2026
**Актуализация:** 22.05.2026
**Цель:** Подтверждение работоспособности облегчённого стека (Qdrant + FastAPI + LlamaIndex + MLX Host), SafeRAG/CRAG-пайплайна, split UI и артефактов.

## 1. Общие положения
### 1.1. Объект испытаний
Локальная Единая Система v2.0+ — суверенный инженерный RAG-стек без внешних зависимостей (RAGFlow, ES, MySQL, MinIO). Работает на FastAPI, Qdrant и MLX Host (`Qwen3-14B`, `Qwen3-4B`, `bge-m3`). Ollama оставлен как резерв.

### 1.2. Условия проведения
- Docker Desktop / Compose
- MLX Host на хосте (8080)
- Ollama на хосте (11434), опционально как резерв
- Mac M4 / 24 GB RAM (целевая платформа)
- Тестовые данные: PDF/DOCX нормативы, EML/MSG письма, XLSX таблицы

### 3.2. Заключение
Система Л.Е.С. v2.0 Core демонстрирует стабильную работу основного пайплайна, UI и метрик. Критических ошибок нет. Для ввода в промышленную эксплуатацию необходимо:
1. Провести нагрузочные тесты (latency, swap, параллельные запросы).
2. Внедрить retry-логику и No-Cache заголовки.
3. Проверить парсинг EML/MSG и XLSX на реальных письмах/сметах.
4. Реализовать Folder Watcher для автоматической синхронизации.

Система считается **готовой к опытной эксплуатации** при условии доработки пунктов выше.
