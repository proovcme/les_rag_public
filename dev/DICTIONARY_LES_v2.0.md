# Словарь Акронимов Комплекса «Л.Е.С.» v2.0

| Акроним | Полное название | Роль в системе v2.0 |
|---|---|---|
| Л.Е.С. | Локальная Единая Система | Главная система, оркестратор, закрытая сеть. FastAPI Proxy. |
| Ж.А.Б.А. | Жесткая Аппаратная База Аналитики | Физический фундамент (Mac Mini M4 / 24 GB, ZeroTier, Docker). |
| С.А.М.О.В.А.Р. | Система Автономная Машинной Обработки Внутренних Архивов РАГ | Ядро RAG: Qdrant + LlamaIndex + `bge-m3` + `qwen3:14b`. |
| Т.О.С.К.А. | Терминал Оценки, Самопроверки и Контроля Архитектуры | Нативный CRAG-пайплайн (Pre/Post-Check, валидация цитат, фильтр галлюцинаций). |
| В.О.Л.К. | Внутренний Охранный Локальный Контур | RBAC, JWT-аутентификация, маскирование секретов, контроль доступа. |
| С.О.В.У.Ш.К.А. | Система Обработки и Выдачи: Умная, Шаблонизированная, Классифицированная, Автоматизированная | Интеллектуальный UI: чат, Chart.js-метрики, SSE-логи, табы. |
| К.О.Т. | Куратор Отраслевой Терминологии | Семантический фильтр инженерного языка, pre-check контекста. |
| С.У.Х.А.Р.И.К. | Система Управления Холодными Архивами и Резервными Источниками Комплекса | Снапшоты Qdrant, бэкапы `storage/datasets/`, ротация архивов. |
| П.Р.О.Р.А.Б. | Программа Регулярной Оценки Работы Автономной Базы | Сбор метрик (psutil, latency, CRAG-score), `/api/metrics`, диагностика. |

📅 **Актуализировано:** 10.05.2026 — v2.0 Core (без RAGFlow/ES/MySQL)
